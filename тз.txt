4) useEffect, что делает и зачем нужен - всем
(useEffect — это хук React, который позволяет выполнять побочные эффекты в функциональных компонентах. Он используется во всех приложениях React, включая приложения на Next.js.

Основные моменты о useEffect:
Побочные эффекты:
useEffect позволяет выполнять действия, которые не относятся напрямую к отрисовке компонента, например:
Запросы к API
Подписка на события
Изменение или управление состоянием)


5) useState, что делает и зачем нужен - всем
(useState — это хук React, который позволяет добавить состояние в функциональные компоненты. Он используется во всех приложениях на React, включая те, которые создаются с помощью Next.js. Этот хук позволяет хранить и обновлять значения, которые могут меняться в процессе работы компонента.

Основные моменты о useState:
Инициализация состояния:
useState принимает один аргумент, который является начальным значением состояния. Этот аргумент может быть любого типа: строкой, числом, массивом, объектом и даже функцией.
Возвращаемые значения:
useState возвращает массив, состоящий из двух элементов:
Текущее значение состояния
Функция для обновления этого значения)
